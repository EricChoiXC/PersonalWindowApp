# 桌面管家需求文档

## 一：应用公共功能

- 自启动；启动后自动隐藏桌面上的所有文件，在任务栏生成应用图标；
- 无数据库管理，数据信息本地配置文件保存；
- 在应用图标右键单击，提供菜单栏：
  - 图标
  - --------
  - 退出
- 在应用图标左键点击，触发"图标"菜单功能

## 二：图标管理

1. 应用启动后：自动隐藏桌面上的所有文件；应用运行时，系统在桌面上创建/移入一个文件时，也自动隐藏该文件

2. 触发"图标"菜单，打开一个窗口；窗口内显示所有的桌面文件；

3. 在该窗口内，可以创建/编辑/删除"分组"，分组可以自定义命名、可重复；初始默认只有一个"未命名"分组；所有文件初始默认在这个分组下；分组内需要无文件才允许删除；

4. 可以拖动文件快速移动所属分组；
