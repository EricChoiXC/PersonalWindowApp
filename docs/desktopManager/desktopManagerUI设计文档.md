# 桌面管家 UI 设计文档

## 1. 页面清单

| 页面编号 | 页面名称 | 类型 | 说明 |
|---------|---------|------|------|
| P01 | 系统托盘菜单 | 托盘右键菜单 | 应用入口，最小化后通过托盘操作 |
| P02 | 图标管理窗口 | 主窗口 | 桌面文件列表与分组管理 |

## 2. P01 - 系统托盘菜单

### 2.1 页面元信息

- **页面标识**：`SystemTray`
- **窗口类型**：AWT PopupMenu（系统托盘右键菜单）
- **打开方式**：应用启动后自动创建，右键单击托盘图标弹出
- **关闭方式**：点击菜单外部区域自动关闭

### 2.2 页面目标

- 提供应用快捷操作入口
- 不占桌面空间，随应用常驻

### 2.3 菜单结构

```
┌──────────────────────┐
│ 🖥 图标              │  ← 左键单击 / 右键首项
├──────────────────────┤
│ 🚪 退出              │
└──────────────────────┘
```

- **图标**：打开 P02 图标管理窗口
- **分隔线**：视觉分隔
- **退出**：退出整个应用，恢复桌面图标显示

### 2.4 交互约束

- 左键单击托盘图标：等同于点击"图标"菜单项，打开图标管理窗口
- 右键单击托盘图标：弹出上述菜单
- "退出"点击后：
  1. 恢复桌面图标为可见状态
  2. 停止文件监控线程
  3. 保存当前配置
  4. 移除托盘图标
  5. 退出 JVM

## 3. P02 - 图标管理窗口

### 3.1 页面元信息

- **页面标识**：`IconManageWindow`
- **窗口类型**：JavaFX Stage（非模态）
- **FXML 文件**：`IconManage.fxml`
- **Controller**：`IconManageController`
- **最小尺寸**：640×480
- **默认尺寸**：800×600
- **打开方式**：托盘菜单"图标"或左键单击托盘图标

### 3.2 页面目标

- 展示当前桌面所有文件
- 支持按分组查看文件
- 提供分组创建、编辑、删除功能
- 支持拖拽文件切换分组

### 3.3 页面布局

```
┌─────────────────────────────────────────────────────────┐
│ 桌面管家 - 图标管理                              [─] [□] [×] │
├──────────────┬──────────────────────────────────────────┤
│  分组         │  文件列表（分组名：未命名）                  │
│              │                                          │
│ ┌──────────┐ │  ┌────┬──────────────────────────────┐   │
│ │ 🗂 未命名 │ │  │ 🗂│ 📄 report.xlsx               │   │
│ ├──────────┤ │  ├────┼──────────────────────────────┤   │
│ │ 🗂 工作   │ │  │ 🗂│ 📝 notes.txt                 │   │
│ ├──────────┤ │  ├────┼──────────────────────────────┤   │
│ │          │ │  │ 🗂│ 🖼 screenshot.png             │   │
│ │          │ │  ├────┼──────────────────────────────┤   │
│ │          │ │  │ 🗂│ ...更多文件                   │   │
│ │          │ │  └────┴──────────────────────────────┘   │
│ │          │ │                                          │
│ ├──────────┤ │  共 12 个文件                              │
│ │ [+ 新建] │ │                                          │
│ └──────────┘ │                                          │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

### 3.4 结构树与布局约束

```
IconManageWindow (BorderPane)
├─ left: 分组面板 (VBox, prefWidth=180)
│   ├─ Label "分组" (header, bold)
│   ├─ Separator
│   ├─ ListView<GroupVo> (fx:id="groupListView", VBox.vgrow=ALWAYS)
│   │   └─ 每个 Item: 分组名称 + 右键菜单
│   └─ HBox
│       └─ Button "＋ 新建分组" (fx:id="addGroupButton")
│
└─ center: 文件面板 (VBox)
    ├─ HBox (header)
    │   ├─ Label 当前分组名称 (fx:id="currentGroupLabel", 动态)
    │   └─ Region (HBox.hgrow=ALWAYS, 占位)
    ├─ Separator
    ├─ ListView<DesktopFileVo> (fx:id="fileListView", VBox.vgrow=ALWAYS)
    │   └─ 每个 Item: 文件图标 + 文件名 + 文件类型标签
    └─ HBox (footer)
        ├─ Label 文件计数 (fx:id="fileCountLabel")
        └─ Region (HBox.hgrow=ALWAYS, 占位)
```

### 3.5 字段与组件规格

#### 3.5.1 分组面板（左侧）

| 组件 | fx:id | 类型 | 说明 |
|------|-------|------|------|
| 分组标题 | — | Label | 固定文本"分组" |
| 分组列表 | `groupListView` | `ListView<GroupVo>` | 显示所有分组，当前选中高亮 |
| 新建分组按钮 | `addGroupButton` | Button | 点击弹出新建分组输入框 |

#### 3.5.2 文件面板（右侧）

| 组件 | fx:id | 类型 | 说明 |
|------|-------|------|------|
| 当前分组名 | `currentGroupLabel` | Label | 显示"分组名：{名称}" |
| 文件列表 | `fileListView` | `ListView<DesktopFileVo>` | 显示当前分组下的文件 |
| 文件计数 | `fileCountLabel` | Label | 显示"共 N 个文件" |

#### 3.5.3 分组列表项

每个分组项右键菜单：

```
┌──────────────┐
│ ✏ 重命名     │
├──────────────┤
│ 🗑 删除       │  ← 分组内无文件时可用，否则灰色
└──────────────┘
```

#### 3.5.4 文件列表项

每个文件项显示：

```
┌──────────────────────────────────────────────┐
│ [系统文件图标]  文件名.扩展名                  │
└──────────────────────────────────────────────┘
```

- 文件图标：根据文件扩展名显示对应的系统图标
- 文件名：完整文件名（含扩展名）
- 支持拖拽（见 3.6）

### 3.6 操作与状态约束

#### 3.6.1 分组操作

| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 选中分组 | 单击分组列表项 | 右侧加载该分组下的文件列表 |
| 新建分组 | 点击"+ 新建分组" | 弹出对话框，输入分组名，确认后创建并选中 |
| 重命名分组 | 右键 → "重命名" | 弹出对话框，预填当前名称，修改后确认 |
| 删除分组 | 右键 → "删除" | 分组内有文件时禁用该菜单项；为空时二次确认后删除 |
| 默认空状态 | — | 初始只有一个"未命名"分组，该分组不可删除 |

#### 3.6.2 拖拽操作

| 操作 | 行为 |
|------|------|
| 拖起 | 从文件列表项开始拖拽（DragDetected） |
| 拖过 | 拖拽经过分组列表项时高亮目标分组（DragOver） |
| 放下 | 将文件从源分组移动到目标分组（DragDropped），更新配置并刷新两侧列表 |

- 拖拽同分组内放下：无变化
- 拖拽到分组列表空白区域：无变化
- 拖拽视觉反馈：源文件半透明，目标分组边框高亮

#### 3.6.3 窗口状态

| 状态 | 说明 |
|------|------|
| 正常显示 | 窗口打开，文件列表可见 |
| 关闭 | 点击关闭按钮，窗口隐藏（不退出应用），保留当前选中状态 |
| 再次打开 | 恢复上次关闭时的分组选中状态 |
| 文件为空 | 当分组下无文件时，列表显示"此分组下暂无文件"提示文字 |

## 4. 弹窗规格

### 4.1 新建/重命名分组弹窗

```
┌───────────────────────────┐
│ 新建分组             [×] │
├───────────────────────────┤
│                           │
│ 分组名称: [____________]  │
│                           │
│       [取消]  [确定]      │
└───────────────────────────┘
```

- **窗口类型**：JavaFX Stage（APPLICATION_MODAL）
- **尺寸**：350×150（固定）
- **输入框**：非空校验，最大 20 字符
- **确定按钮**：名称不重复时创建，否则提示"分组名称已存在"
- **取消按钮**：关闭弹窗不保存

### 4.2 确认删除弹窗

```
┌───────────────────────────────┐
│ 确认删除                 [×] │
├───────────────────────────────┤
│                               │
│ 确定要删除分组"XXX"吗？       │
│ 此操作不可恢复。              │
│                               │
│         [取消]  [确定删除]    │
└───────────────────────────────┘
```

- **窗口类型**：JavaFX Stage（APPLICATION_MODAL）
- **尺寸**：350×160（固定）

## 5. 样式说明

- 默认使用浅色主题，主色调为 Windows 系统风格
- CSS 文件：`main-style.css`
- 使用 `.group-item`、`.group-item-selected`、`.file-item` 等样式类
- 窗口标题栏使用系统默认风格
- 图标使用 Unicode 或系统默认图标，不依赖外部图片资源

## 6. 窗口交互流程

```
应用启动
  │
  ├─ 隐藏桌面图标
  ├─ 创建系统托盘
  └─ 启动文件监控
        │
        ▼
  系统托盘图标（常驻）
  ├─ 左键 → 打开/激活图标管理窗口
  └─ 右键 → 弹出菜单
               ├─ "图标" → 打开/激活图标管理窗口
               └─ "退出" → 恢复桌面 + 保存配置 + 退出

图标管理窗口
  ├─ [×] 关闭 → 隐藏窗口（不退出应用）
  ├─ 点击分组 → 切换文件列表
  ├─ 新建分组 → 弹窗 → 确定 → 刷新列表
  ├─ 重命名分组 → 弹窗 → 确定 → 刷新列表
  ├─ 删除分组 → 确认弹窗 → 确定 → 刷新列表
  └─ 拖拽文件 → 切换分组 → 刷新列表 + 保存配置
```
